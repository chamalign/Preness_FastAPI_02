#6 Design requirements by question type
(Main Idea / Factual / Inference / Vocabulary / Usage / NOT / Rhetorical / Organization / Location / Viewpoint)

1) Main Idea
Objective: measure ability to integrate the full text into one abstract proposition.
Design requirements (high precision):
- Correct option summarizes the function/claim of the whole passage at medium abstraction (30-50%).
- Do not include proper nouns, numbers, examples, or paragraph-specific details.
- Focus on what the author does in the passage, not just the topic label.
Prohibited:
- Summary of only one specific paragraph
- Direct paraphrase of the conclusion sentence
- Option that only states the topic name

2) Factual
Objective: measure accurate retrieval and recognition of explicit information.
Design requirements (high precision):
- Correct option must be explicitly stated in one sentence or adjacent two sentences.
- Use stems such as according to / states / indicates.
- Do not include inference, evaluation, or generalization.
Prohibited:
- Information synthesized from multiple distant locations
- Mixed fact + interpretation options

3) Inference
Objective: measure ability to infer one logical step beyond explicit text.
Design requirements (high precision):
- Correct option must be inferred from two or more textual facts.
- Conclusion should be modest and limited (weak inference).
- Must not contradict the passage or require extra knowledge.
Prohibited:
- Added common/background knowledge
- Strong absolutes (always / must / never)
- Causal leap or reversal

4) Vocabulary
Objective: measure context-based meaning selection.
Design requirements (high precision):
- Target words should be B1-B2 polysemous/abstract/figurative expressions.
- Correct option must naturally replace the word in context.
- Must require context reading to solve.
Prohibited:
- Simple synonym-only test
- Solvable by first dictionary meaning alone

5) Usage
Objective: measure accurate processing of referential relations in context.
Design requirements (high precision):
- Antecedent should generally be within the previous 1-2 sentences.
- Must be uniquely identifiable by number, abstract/concrete fit, and logical consistency.
- In explanation and wrong_reason_*, state the referent location using "line ○" or "○文目". Do NOT use "In passage 〇" or "paragraph 〇".
Prohibited:
- Solvable by grammar knowledge alone
- Ambiguous reference with multiple plausible antecedents
- Restricting prompts only to they/them/theirs/it, etc.

6) NOT
Objective: measure global comprehension and elimination strategy.
Design requirements (high precision):
- Only the correct option is not present in the passage.
- All other options must be text-supported (paraphrase allowed).
- Passage design for NOT: The three mentioned options (distractors) must be supported by one sentence, or two adjacent sentences, in the passage. Do NOT spread them across the whole passage or multiple paragraphs (e.g. avoid comma-separated lists scattered over many lines). The question should be answerable by reading that single sentence or adjacent pair.
Prohibited:
- Correct option that is only partially different
- NOT question requiring inference
- NOT items where the three mentioned options are scattered across the entire passage (must be localized to 1–2 sentences).

7) Rhetorical
Objective: measure understanding of rhetorical/logical function.
Design requirements (high precision):
- Target functions such as example, contrast, definition, or elaboration.
- Must be uniquely determined from local context.
Prohibited:
- Speculation about author emotions/psychology
- Confusion with organization-type questions

8) Organization
Objective: measure meta-level understanding of text structure.
Design requirements (high precision):
- Assume clear structural progression (general->specific, cause->effect, etc.).
- Ask about structure itself, not details.
Prohibited:
- Detail-specific options
- Overlap with main-idea questions

9) Location
Objective: measure ability to locate where information appears.
Design requirements (high precision):
- Target information must be uniquely locatable.
- Paraphrase should still allow clear location mapping.
Prohibited:
- Design that matches multiple locations
- Ambiguous keywording

10) Viewpoint / Tone
Objective: measure understanding of evaluative stance in academic prose.
Design requirements (high precision):
- Base judgment on tone, hedging, and qualifiers.
- Correct options include neutral / cautious / objective / mildly critical, etc.
Prohibited:
- Strong emotional wording
- Subjective value judgment
Do not generate or adopt any item that fails any requirement above.

#7 Underlying item-design philosophy
- TOEFL ITP Reading measures text-based information processing, not outside knowledge or common sense.
- Correct answers are always accurate paraphrases of passage content, not identical wording.
- Each question type targets a distinct reading skill.
- Distractors should reflect typical errors: misreading, logical leap, and scope misinterpretation.
- Vocabulary questions measure context-based meaning selection, not raw vocabulary size.
- Inference questions must accept only modest one-step logical conclusions from the text.
- Tone/viewpoint questions assess stance (neutral/cautious/objective), not emotion.
- All questions must be answerable from passage information alone, without cultural/specialized background knowledge.
- TOEFL ITP Reading is reproducible: ratio, structure, and question form remain consistent.

(Premise 2: Answer-choice design)
#1 Common rules
- Each question has 4 options.
- Correct option is a semantically accurate paraphrase of the passage.
- Distractors must be grammatically/semantically natural but inconsistent with the passage.
- Do not create unnatural length gaps between correct and incorrect options.

#2 Option-design requirements by question type
- Main Idea:
  - Average option length: 6-12 words
  - Correct option definition:
    - Summarize the whole passage at medium abstraction (30-50%).
    - No concrete examples, numbers, or names.
    - Reflect author action (explain/describe/compare/argue).
  - Distractor definitions:
    - Detail-only type: only one paragraph/example
    - Topic-label type: only topic name, no claim
    - Over-abstract type: too broad and non-specific
    - Distortion type: overstate or understate author stance

- Factual:
  - Average option length: 7-14 words
  - Correct option definition:
    - Explicitly stated in the passage
    - Verifiable in one sentence or adjacent two sentences
    - Paraphrase limited to wording/syntax level
  - Distractor definitions:
    - One-word difference type (number/degree term)
    - Mixed-source type (combines two passage points)
    - Near-negation type (almost/often/usually, etc.)
    - Out-of-passage type (plausible but unmentioned)

- Inference:
  - Average option length: 8-16 words
  - Correct option definition:
    - Inferable from two or more textual facts
    - Modest and limited (may/likely)
    - Does not contradict the passage
  - Distractor definitions:
    - Overgeneralization type (always/never)
    - Added world-knowledge type
    - Causal reversal type
    - Under-supported type (insufficient information)

- Vocabulary:
  - Average option length: 1 word
  - Correct option definition:
    - Naturally replaceable in context
    - Context-dependent meaning
    - One intended sense of a polysemous/abstract word
  - Distractor definitions:
    - Dictionary-first-meaning type
    - Partial-match type
    - Part-of-speech mismatch type
    - Positive/negative polarity reversal type
  - Adverbs, adjectives, and abstract nouns are valid targets.

- Usage:
  - Average option length: 2-6 words
  - Correct option definition:
    - Exactly one valid referent in context
    - Number, meaning, and logic all align
  - Distractor definitions:
    - Nearby-noun trap
    - Number-mismatch trap
    - Abstract/concrete mismatch trap
    - Grammatically possible but semantically inconsistent trap

- NOT:
  - Average option length: 7-14 words
  - Correct option definition:
    - Not present anywhere in the passage
    - Still plausible in content
  - Distractor definitions:
    - Only one option may be outside the passage
    - "Partially wrong" correct option is not allowed

- Rhetorical:
  - Average option length: 7-14 words
  - Correct option definition:
    - Captures logical function (example/contrast/definition/elaboration)
    - Uniquely determined from nearby context
  - Distractor definitions:
    - Emotion-guessing type
    - Content-summary type
    - Organization-confusion type
    - Over-attributed-purpose type

- Organization:
  - Average option length: 6-14 words
  - Correct option definition:
    - About structure, not details
    - General->specific, cause->effect, etc.
  - Distractor definitions:
    - Detail-content type
    - Paragraph-limited type
    - Main-idea paraphrase type
    - Timeline-misread type

- Location:
  - Average option length: 3-8 words
  - Correct option definition:
    - Uniquely locatable in the passage
    - Clear paragraph/position reference
  - Distractor definitions:
    - Multi-location ambiguity type

- Viewpoint / Tone:
  - Average option length: 1-3 words
  - Correct option definition:
    - neutral / cautious / objective / mildly critical
    - Based on tone and qualifiers
  - Distractor definitions:
    - Emotion-heavy type
    - Extreme evaluation type
    - Exaggerated-claim type
    - Subjective value-judgment type

Additional notes:
- Include qualifiers in correct options:
  - tends to
  - in some cases
  - largely
  - primarily
  - etc.
- Distractors should look grammatical and initially plausible, but be misaligned on close reading.
  - Overscope type
  - Causal-direction mismatch type
  - Cross-paragraph synthesis mismatch type
Use the above to diversify distractor patterns.
